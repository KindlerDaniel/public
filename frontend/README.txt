In einem typischen (z. B. React-)Frontend enthält public/ statische Dateien wie index.html, Bilder, Icons 
oder andere Ressourcen, die direkt vom Webserver ausgeliefert werden, 
ohne dass sie noch durch das Build-Tool verarbeitet werden.

package.json ist das zentrale Konfigurations- und Steuerungs-File für ein Node.js-basiertes Frontend-Projekt.
Sie definiert Metadaten zum Projekt (Name, Version usw.)
Listet die benötigten Abhängigkeiten (Dependencies) auf
Enthält Skripte, die über npm oder yarn ausgeführt werden können 
(z. B. npm run start, npm run build)
